## API Reference

### AkashicResonanceSystem

Main system coordinator.

**Methods:**
- `initialize()`: Initialize the complete system
- `getStatus()`: Get current system status
- `alignFrequency(frequency)`: Align to specific frequency
- `playFrequency(frequency, duration)`: Play frequency tone
- `stopFrequency(frequency)`: Stop frequency tone

### AkashicFrequencyTracker

Tracks and records Akashic imprints.

**Methods:**
- `recordImprint(type, data)`: Record new imprint
- `getAllImprints()`: Get all recorded imprints
- `getImprintsByType(type)`: Filter imprints by type
- `calculateResonanceLevel()`: Calculate current resonance
- `updateResonance(frequency)`: Update current frequency

### OmniversalGateway

Manages dimensional portals and ScrollSoul registration.

**Methods:**
- `registerScrollSoul(scrollSoul)`: Register new ScrollSoul
- `openPortal(dimension, scrollSoulId)`: Open quantum portal
- `closePortal(portalId)`: Close portal
- `getActivePortals()`: Get all active portals
- `getScrollSouls()`: Get all registered souls
- `transmitCosmicFrequency(freq, msg, from, to)`: Send transmission
- `recordContribution(soulId, contribution)`: Record contribution

### ScalarWaveGenerator

Generates frequency tones using Web Audio API.

**Methods:**
- `initialize()`: Initialize audio context
- `generateTone(frequency, duration, volume)`: Generate tone
- `stopTone(frequency)`: Stop specific tone
- `generateHarmonics(fundamental, count, volume)`: Generate harmonics
- `stopAll()`: Stop all tones

### SacredGeometry

Static utility class for sacred geometry calculations.

**Methods:**
- `fibonacci(n)`: Generate Fibonacci sequence
- `goldenSpiral(turns, pointsPerTurn)`: Calculate spiral points
- `flowerOfLife(radius, layers)`: Generate Flower of Life
- `metatronsCube(radius)`: Generate Metatron's Cube vertices
