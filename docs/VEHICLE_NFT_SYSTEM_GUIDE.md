## Integration Guide

### Quick Start

```javascript
const { VehicleOrchestrator } = require('./src/vehicles/index.js');

// Initialize the system
const orchestrator = new VehicleOrchestrator({ frequency: 963 });
await orchestrator.initialize();
await orchestrator.deploy();

// Mint a vehicle NFT with full integration
const userData = {
  vehicleCount: 1,
  totalValue: 50000,
  sustainabilityScore: 0.8,
  hasTwin: false
};

const result = await orchestrator.mintVehicleNFT(
  'ZEEKR_9X',
  '0xBuyerAddress',
  userData
);

console.log('Minted NFT:', result.mint.tokenId);
console.log('Incentives:', result.incentives);
console.log('Pricing:', result.pricing);
```

### Creating Matching Sets

```javascript
// Create a dealer matching set
const vehicleKeys = ['ZEEKR_9X', 'HONGQI_EHS9', 'IM_LS6'];
const dealerData = {
  vehicleCount: 3,
  totalValue: 180000,
  sustainabilityScore: 0.85
};

const setResult = await orchestrator.createMatchingSet(
  vehicleKeys,
  '0xDealerAddress',
  dealerData
);

console.log('Matching Set ID:', setResult.matchingSet.setId);
console.log('Total Value:', setResult.matchingSet.totalValue);
console.log('Lifetime Profit:', setResult.matchingSet.lifetimeProfitPotential);
```

### Participating in Experiments

```javascript
const { IncentivesSystem } = require('./src/vehicles/index.js');

const incentives = new IncentivesSystem();
await incentives.initialize();

// Participate in NFT staking experiment
const stakingResult = incentives.participateInExperiment('NFT_STAKING', {
  stakedValue: 50000,
  stakingDays: 90
});

console.log('Staking Rewards:', stakingResult.rewards);
console.log('APY:', stakingResult.apy);
```

## API Reference

### VehicleNFTModel

#### Methods

**`initialize()`**
- Initializes the vehicle NFT system
- Returns: `Promise<boolean>`

**`deploy()`**
- Deploys the NFT contract and systems
- Returns: `Promise<Object>`

**`mintVehicleNFT(vehicleKey, ownerAddress)`**
- Mints a new vehicle NFT
- Parameters:
  - `vehicleKey`: String (e.g., 'ZEEKR_9X')
  - `ownerAddress`: String (blockchain address)
- Returns: `Promise<Object>`

**`getVehicleCollection(vehicleKey)`**
- Gets vehicle data by key
- Returns: `Object`

**`getNFTMetadata(vehicleKey)`**
- Gets NFT metadata by vehicle key
- Returns: `Object`

**`getStatistics()`**
- Gets system statistics
- Returns: `Object`

### TechPartnerships

#### Methods

**`initialize()`**
- Initializes technology partnerships
- Returns: `Promise<boolean>`

**`syncWithPartner(partnerKey, vehicleNFTData)`**
- Syncs NFT data with partner
- Returns: `Promise<Object>`

**`createMatchingSet(vehicleKeys, dealerAddress)`**
- Creates a dealer matching set
- Returns: `Promise<Object>`

**`getPricingForVehicle(vehicleData)`**
- Gets pricing information
- Returns: `Object`

**`getCompatiblePartners(vehicleKey)`**
- Gets compatible partners for vehicle
- Returns: `Array<Object>`

### IncentivesSystem

#### Methods

**`initialize()`**
- Initializes incentives system
- Returns: `Promise<boolean>`

**`calculateIncentives(userData, vehicleData)`**
- Calculates user incentives
- Returns: `Object`

**`applyPricingBoost(basePrice, userData)`**
- Applies pricing discount based on tier
- Returns: `Object`

**`participateInExperiment(experimentKey, participantData)`**
- Participates in market experiment
- Returns: `Object`

**`trackTwinAdoption(region, adoptionType, vehicleKey)`**
- Tracks twin adoption by region
- Returns: `Object`

**`calculateSustainabilityScore(userData)`**
- Calculates sustainability score
- Returns: `Number` (0-1)

### VehicleOrchestrator

#### Methods

**`initialize()`**
- Initializes all subsystems
- Returns: `Promise<boolean>`

**`deploy()`**
- Deploys all subsystems
- Returns: `Promise<Object>`

**`mintVehicleNFT(vehicleKey, ownerAddress, userData)`**
- Mints NFT with full integration
- Returns: `Promise<Object>`

**`createMatchingSet(vehicleKeys, dealerAddress, userData)`**

## Best Practices

### 1. Initialization Sequence

Always initialize in the correct order:
1. VehicleNFTModel
2. TechPartnerships
3. IncentivesSystem
4. Deploy all systems

### 2. User Data Management

Provide complete user data for accurate tier calculation:
```javascript
{
  vehicleCount: Number,
  totalValue: Number,
  sustainabilityScore: Number (0-1),
  hasTwin: Boolean,
  // Sustainability factors (optional)
  carbon_footprint: Number,
  renewable_energy: Number,
  battery_lifecycle: Number,
  material_sourcing: Number,
  circular_economy: Number
}
```

### 3. Error Handling

Always wrap operations in try-catch blocks:
```javascript
try {
  const result = await orchestrator.mintVehicleNFT(vehicleKey, address, userData);
  // Handle success
} catch (error) {
  // Handle error
  console.error('Minting failed:', error.message);
}
```

### 4. Statistics Monitoring

Regularly check system statistics:
```javascript
const stats = orchestrator.getStatistics();
console.log('Active Tokens:', stats.vehicleNFT.activeTokens);
console.log('Active Syncs:', stats.techPartnerships.activeSyncs);
console.log('Experiments Running:', stats.incentives.experimentsRunning);
```
